# Core Animation and Kite Compositor Tutorials

Here, you will learn from gif-guided tutorials about core animation and Kite Compositor. With these gif-guided tutorials, my intention is to make the power of the Core Animation Framework easy and accessible to non-coders through the most advanced and power UI animation and prototyping tool, Kite Compositor. If you enjoyed these tutorials, I would like you to subscribe to my [Youtube channel](https://www.youtube.com/nimbbble) or support my [Patreon](https://www.patreon.com/nimbbble) to produce more gif-guided and video tutorials to make the power of Core Animation and Kite Compositor accessible to everyone..
